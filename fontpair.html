Font Pair

Website Design, Website Development

While working with web fonts for our website projects we realized there was no easy way to find Google Font pair inspiration. We were also tired of downloading web fonts in order to test them out together in pairs. We created Font Pair in order to help designers and developers pair better Google fonts together.
